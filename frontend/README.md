# List App

This a fullstack web application that lets you keep notes and deadlines.

## Features

- Google sign in with AOuth 2.0,
- OTP Code for email validation,
- Rate limit for preventing dos attacks.
- Authentication with JWT standard and Cookies.

## Technologies used

- Backend: express.js,
- Frontend: React + Vite,
- Styling: Tailwind, lucid react, daisyui
- Database: noSQL Mongodb with mongoose library,
